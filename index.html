원고 수정 Gems LIVE

Gemini 기반 원고 수정 에이전트. 인터뷰 원고의 문맥을 이해하고 자연스러운 문장으로 다듬어줍니다. Google Gemini Gems에서 바로 실행됩니다.
→ 🎬

영상 길이 예측기 LIVE

인터뷰 원고의 타임스탬프와 학습 데이터를 기반으로 최종 영상 길이를 예측합니다. Word 삭제선을 감지하여 삭제 구간과 잔존 분량을 자동 계산합니다.
→ 💬

강조자막 생성기 LIVE

강조자막 생성 에이전트. 2-Pass 시스템(Draft→Editor)으로 핵심 발언을 선별하고, 16가지 자막 유형을 자동 분류합니다. 형광펜 매칭과 수동 추가를 지원합니다.
→ 📊

자료 & 그래픽 생성기 LIVE

시각화 & 인서트컷 가이드 에이전트. 21가지 템플릿(Chart.js, HTML/CSS, SVG)으로 시각화 가이드를 생성하고, 일러스트·공식이미지·실물 3가지 유형의 인서트컷을 추천합니다.
→ 🎬

영상 수정사항 LIVE

유튜브 영상의 수정 요청을 타임코드 기반으로 전달합니다. 리뷰어가 영상을 보며 수정 포인트를 기록하고, 편집자가 완료 체크로 진행 상황을 추적합니다.
→ ✂️

하이라이트 편집기 LIVE

인터뷰 원고에서 30~40초 분량의 하이라이트 구간을 선택·조합합니다. AI가 임팩트 있는 발언을 추천하고, 편집자가 드래그로 순서를 조정하여 하이라이트를 구성합니다.
→ 📦

세트 생성기 LIVE

인터뷰 원고에서 썸네일 제목, 유튜브 제목, 설명문, 태그를 AI가 생성합니다. 실시간 트렌드를 반영한 3가지 성격(밸런스/스크립트/트렌드)의 후보를 개별 생성합니다.
→ 🎯

삭제선 변환기 LIVE

취소선 → 삭제선 변환 에이전트. Word 문서의 취소선(검토 모드) 표시를 추적하여 삭제선으로 변환하고, 편집 전후 원고의 분량 변화를 분석합니다.
→ 📝

ttimes doctor LIVE

인터뷰 원고 1차 교정 에이전트. STT 오인식 용어 교정, 추임새 제거, 맞춤법 교정을 수행하고 스크립트 편집 기능을 제공합니다. 팀 공유 단어장으로 일관된 용어 관리가 가능합니다.
